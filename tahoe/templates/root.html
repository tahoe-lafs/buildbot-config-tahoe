Welcome to the Tahoe-LAFS Buildbot!

See also the zfec buildbot and the pycryptopp buildbot.

  • Supported Builders -- all the builders that we expect to keep green. If one of these reports a failure, we fix it right away. Whenever a patch is committed all of the test builders are run, and if all of the Supported test builders pass tests then the package-creation builders are run. See the BuildbotPolicy page on the wiki.
  • Unsupported Builders -- the builders that we aren't currently committing to keep green. Hopefully they will level-up to Supported Builders in the future, especially if you offer to help.
